nr-web examples

Small standalone pages that share common.js, relay helpers, and modal fragments with the main app. Use them as patterns for your own static Nostr front-ends.

  • Pixel — circle-scoped 16×16 grid, publish to relays, optional “where / jam” meta.
  • Main app — map + chats in one page (index.html).

See README.md in this folder (on GitHub) for notes on paths, static hosting, and forking.
