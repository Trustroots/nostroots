nr-web examples

Small standalone pages that mirror the main app's chrome (Keys/Settings modals, relay helpers, hash router). The main app is now shipped as just index.html + index.js; copy the NR_COMMON_JS_BEGIN / NR_HASH_ROUTER_BEGIN blocks out of index.html if you need the same helpers in a fork.

  • Pixel — circle-scoped 16×16 grid, publish to relays, optional “where / jam” meta.
  • Main app — map + chats in one page (index.html).

See README.md in this folder (on GitHub) for notes on paths, static hosting, and forking.
