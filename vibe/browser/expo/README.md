# Nostroots Browser (Expo prototype)

This directory is **probably deprecated**. The primary Nostroots Browser app lives in [`../ios/`](../ios/) (native SwiftUI + WKWebView).

We are keeping this Expo prototype in the tree for now until there is a **native Android** app with equivalent NIP-07 and browser behavior. Do not treat it as the source of truth for product or release work.

Run `pnpm install` from the repo root (package is `vibe/browser/expo` in the workspace). Prefer [`../ios/`](../ios/) for day-to-day browser work.
