Welcome to Nostroots

Nostroots is an attempt to rebuild Trustroots on open protocols so travelers and hosts keep more power: less should ride on whoever runs the core Trustroots service, and anyone should be able to build new applications on the same trust network—more freedom for users and other developers alike.

Two ways to get started: import an existing private key, or generate a new one below.

Option 1: Import an existing key

If you already have a Nostr private key (nsec), paste it below (for example from the Nostroots mobile app or your password manager).

npub1… is your public id (like a username). It cannot log you in here. Use nsec1… or your recovery phrase — the same secret you would use to restore the app in a new install.

Option 2: Generate a new key

If you do not have one yet, generate a key below, then export and store your nsec in your password manager. If you do not have a password manager yet, we recommend Bitwarden.com.
