About Squatbridge

Squatbridge bridges event listings into Nostr from the Trustroots Nostr web hub (#237). It pulls squat and autonomous-space events from radar.squat.net and lets you browse them on a map.

  • Pick a city or country — URLs like #city/Berlin are shareable.
  • The list shows today and upcoming events; filter by keyword or reveal past ones.
  • The map keeps markers from cached locations (up to 7 days; event lists stay fresh for 24 h).
  • Share to Nostr — map notes and optional “I'm going” RSVP.

Note format (for other Nostr apps)

Shared events are kind 30397 map notes so the Trustroots validation service can repost them as kind 30398 for clients such as nr-app. To render them directly, read content for the human text and the tags below for map placement, links, and filters.

Content (content)

Newline-separated plain text, in order:

  1. Event title
  2. Date/time as yyyy-mm-dd hh:mm (same-day ranges: 2026-06-08 16:00 → 18:00)
  3. Venue name
  4. Categories (comma-separated), if any
  5. Canonical radar.squat.net URL
  6. Optional #circle hashtags when Trustroots circles are selected

Tags

  • ["d", "squatbridge:radar:{nid}"] — stable id (NIP-33); dedupe on this.
  • ["r", "{url}"] — source link (NIP-01).
  • ["linkLabel", "radar.squat.net"] — short label for the link in map UIs.
  • ["L","open-location-code"] + ["l","{plus}","open-location-code"] — pin on map (Open Location Code).
  • ["L","open-location-code-prefix"] + one ["l","{prefix}","open-location-code-prefix"] per zoom level — area index for map queries.
  • ["L","trustroots-circle"] + ["l","{slug}","trustroots-circle"] — optional circle filters (lowercase slug).
  • ["expiration", "{unix}"] — hide after event end (+1 day) or +7 days if open-ended.

Going to the event (optional, NIP-52)

When the share toggle is on, Squatbridge also publishes a NIP-52 calendar event (kind 31923 or 31922) plus an RSVP (kind 31925) with ["status", "accepted"] and ["fb", "busy"]. The RSVP a tag points at {kind}:{pubkey}:squatbridge:radar:{nid}.

{ "kind": 30397, "content": "Workshop title\n2026-06-07 14:00\nVenue name\nworkshop\nhttps://radar.squat.net/en/node/123456\n#punks", "tags": [ ["d", "squatbridge:radar:123456"], ["r", "https://radar.squat.net/en/node/123456"], ["linkLabel", "radar.squat.net"], ["L", "open-location-code"], ["l", "9F4MFXX6+XX", "open-location-code"], ["L", "open-location-code-prefix"], ["l", "9F4M0000+", "open-location-code-prefix"], ["L", "trustroots-circle"], ["l", "punks", "trustroots-circle"], ["expiration", "1780844400"] ] }

Filter kind 30397 with #l on open-location-code-prefix values for map tiles, or read validated kind 30398 reposts from the Trustroots validation pubkey. Trustroots Map already renders this shape; other clients can treat content as a note body and r as the outbound link.

You need a NIP-07 browser signer (e.g. nos2x, Alby) to publish. Sharing is manual — nothing is posted automatically. Source: vibe/web/examples/squatbridge.html

Publishing to

    Nostr keys

    A Nostr key is your account for Nostr apps. It lets apps recognize you without a password or central account.

    You have a public key, which is safe to share, and a private key, which proves you are you. Keep the private key secret.

    A signer is an app or browser extension that holds your private key and approves actions for you. Nostroots can ask it for your public key, but your private key stays with the signer.

    • On desktop, install the Nostroots Extension from the Chrome Web Store. Works in Chrome, Brave, and Edge. Firefox is in review and coming soon.
    • On mobile, install the Nostroots app for Android or iOS.
    • Reload this page after the signer is available.
    • Connect your Trustroots identity so Nostroots can link your profile and Nostr key.
