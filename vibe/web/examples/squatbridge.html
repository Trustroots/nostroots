About Squatbridge

Squatbridge is an experimental demo from the Trustroots Nostr web hub. It pulls squat and autonomous-space events from radar.squat.net and lets you browse them on a map.

  • Pick a city or country — URLs like #city/Berlin are shareable.
  • The list shows today and upcoming events; filter by keyword or reveal past ones.
  • The map keeps markers from cached locations (up to 7 days; event lists stay fresh for 24 h).
  • Share to Nostr — see note format below.

Note format (for other Nostr apps)

Shared events are kind 30399 third-party map notes, matching the Trustroots Events schema. To render them, read content for the human text and the tags below for map placement, links, and filters.

Content (content)

Newline-separated plain text, in order:

  1. Event title
  2. Date/time (start, or start → end)
  3. Venue name
  4. Categories (comma-separated), if any
  5. Canonical radar.squat.net URL
  6. Optional #circle hashtags when Trustroots circles are selected

Tags

  • ["d", "squatbridge:radar:{nid}"] — stable id (NIP-33); dedupe on this.
  • ["r", "{url}"] — source link (NIP-01).
  • ["linkLabel", "radar.squat.net"] — short label for the link in map UIs.
  • ["L","open-location-code"] + ["l","{plus}","open-location-code"] — pin on map (Open Location Code).
  • ["L","open-location-code-prefix"] + one ["l","{prefix}","open-location-code-prefix"] per zoom level — area index for map queries.
  • ["L","trustroots-circle"] + ["l","{slug}","trustroots-circle"] — optional circle filters (lowercase slug).
  • ["expiration", "{unix}"] — hide after event end (+1 day) or +7 days if open-ended.

Going to the event (optional, NIP-52)

When the share toggle is on, Squatbridge also publishes a NIP-52 calendar event (kind 31923 or 31922) plus an RSVP (kind 31925) with ["status", "accepted"] and ["fb", "busy"]. The RSVP a tag points at {kind}:{pubkey}:squatbridge:radar:{nid}.

{ "kind": 30399, "content": "Workshop title\n2026-06-07T14:00:00+02:00\nVenue name\nworkshop\nhttps://radar.squat.net/en/node/123456\n#punks", "tags": [ ["d", "squatbridge:radar:123456"], ["r", "https://radar.squat.net/en/node/123456"], ["linkLabel", "radar.squat.net"], ["L", "open-location-code"], ["l", "9F4MFXX6+XX", "open-location-code"], ["L", "open-location-code-prefix"], ["l", "9F4M0000+", "open-location-code-prefix"], ["L", "trustroots-circle"], ["l", "punks", "trustroots-circle"], ["expiration", "1780844400"] ] }

Filter kind 30399 with #l on open-location-code-prefix values for map tiles, or subscribe by pubkey. Trustroots Map already renders this shape; other clients can treat content as a note body and r as the outbound link.

You need a NIP-07 browser signer (e.g. nos2x, Alby) to publish. Sharing is manual — nothing is posted automatically. Source: vibe/web/examples/squatbridge.html
